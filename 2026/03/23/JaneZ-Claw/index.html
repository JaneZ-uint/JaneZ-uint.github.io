JaneZ-Claw

其实吧，应该是JaneA-Claw，毕竟JaneA的主人是JaneZ。

配置

如大多数人一样，采用了阿里云服务器配置，首先你需要购买一台阿里云轻量应用服务器，应用镜像设置为openclaw，选择一年的使用期限，由于我错过了入教的黄金时间，价格已由68rmb/Y上调到79rmb/Y，白花11rmb，感觉如果想配还是尽早配，毕竟价格大概还会继续上涨。

接下来的操作都非常傻瓜，你需要等待1-5分钟，直到服务器部署完成，然后一键连通，选择你的API，填入API key，然后就可以获取你的网页端地址了，打开它，你就可以看到xxx-Claw的界面了。

API 购买

本人最近购买了多种不同类型模型的API，包括文本生成模型、图像生成模型和多模态模型，价格从几十到几百不等。由于本人目前对图像生成和多模态模型孰优孰劣还心存疑虑（真神是nano banana pro，但买不起，以及这里穿插一则趣事，lz在使用nano banana pro创作时，众所周知得到的图片会有水印，于是lz找到了一种邪修方法，在你的chrome上下载 Nano Banana Watermark Remover 插件，安装后就可以去除水印了，虽然有点邪修，但效果不错，当然感觉这种严重损害google利益的插件chrome过不了几天就会封掉它🥲），所以先主要分享几款国内文本生成API 使用效果（要想办法搞张境外visa卡了）。

文本生成模型

Deepseek V3.2

  • base_url: https://api.deepseek.com
  • 购买方式：（这是我最喜欢的一点）没有套餐，直接充钱。
  • 价格：反正我没干啥就烧了0.13rmb。
  • 使用效果：一坨，如果你像我一样喜欢干净清楚的支付条例，那你放心用deepseek，史也吃的心甘情愿。

Qwen 3.5-plus

  • base_url: https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions
  • 购买方式：（阿里云的ui简直是一坨，我都不想说了，找了半天没找到在哪儿买，更没看到token使用情况，倒是会时不时的给你发短信提示你还剩多少token记得充钱）套餐制，原文是用于抵扣模型推理超出免费额度后产生的推理费用，20rmb/第一个月，100rmd/前3个月，当然似乎也是有直接充token的选项的，计费规则复杂。
  • 价格：免费送了20万token吧，bot发了两三条消息就烧了1.3万左右。
  • 使用效果：我感觉我还是挺看的起Qwen的，小模型的表现还行。

GLM-5

  • base_url: https://open.bigmodel.cn/api/paas/v4/chat/completions
  • 购买方式：和deepseek一样直接充值，没有套餐（也可能是我没看到）。
  • 价格：实名认证送百万token，感觉还不错。
  • 使用效果：感觉还行，挺快的，但我没怎么用过。
