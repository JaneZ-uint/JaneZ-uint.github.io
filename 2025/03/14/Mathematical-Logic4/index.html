同构
存在一个A -> B 的映射满足：

  1. 是双摄
  2. 保持关系（n元关系运算符）
  3. 保持运算（n元函数运算符）
  4. 保持常量（常数c）

则结构A、结构B同构

在上周的作业中，我们证明了同构是一种等价关系。回忆，等价关系需要满足三个条件：

  1. 自反性
  2. 对称性
  3. 传递性

同构引理的证明

Recall:
A、B是2个同构的S-结构，那么对于所有S-sentence 𝜑，有：

A |= 𝜑 ⟺ B |= 𝜑

Prof:
由重合引理可知：
A |= 𝜑 ⟺ (A,β) |= 𝜑
B |= 𝜑 ⟺ (B,β’) |= 𝜑

so A |= 𝜑 ⟺ B |= 𝜑 ⟺ (A,β) |= 𝜑 ⟺ (B,β’) |= 𝜑

定义β^𝜋(x) := 𝜋(β(x))
那么我们要证：
(A,β) |= 𝜑 ⟺ (B,β^𝜋) |= 𝜑 (𝜑 是任意S-公式)
// 这不就从句子推广到公式上了嘛

也就是要证明：
I:= (A,β) I^𝜋:= (B,β^𝜋)
𝜋(I(t)) = I^𝜋(t)

这里改写成解释，也就可以进行结构归纳法（对公式的归纳了）
