Lemma 8

Let (\varphi) be an (S)-formula and (x_1, \ldots, x_n) pairwise distinct variables. Then

(i)

IОҰвҠЁвҲғx1вҖҰвҲғxnПҶ \mathfrak{I}^\Phi \models \exists x_1 \ldots \exists x_n \varphi IвҖӢОҰвҖӢвҖӢвҠЁвҲғxвҖӢ1вҖӢвҖӢвҖҰвҲғxвҖӢnвҖӢвҖӢПҶ

Consistent

Definition
ОҰ\PhiОҰ is consistent if there is no ПҶ\varphiПҶ such that both ОҰ\PhiОҰ вҠў ПҶ\varphiПҶ and ОҰ\PhiОҰ вҠў В¬\negВ¬ПҶ\varphiПҶ. Otherwise,ОҰ\PhiОҰ is inconsistent.

Negation Complete

Definition
A set (\Phi) is negation complete if for every (S)-formula (\varphi):
ОҰвҠўПҶВ orВ ОҰвҠўВ¬ПҶ. \Phi \vdash \varphi \text{ or } \Phi \vdash \neg \varphi. ОҰвҠўПҶВ orВ ОҰвҠўВ¬ПҶ.

йҖҡдҝ—зҡ„иҜҙпјҢе°ұжҳҜеҰӮжһңОҰ\PhiОҰ иҜҒдёҚеҮәжқҘПҶ\varphiПҶпјҢйӮЈе°ұдёҖе®ҡиғҪиҜҒеҮәжқҘВ¬ПҶ\neg \varphiВ¬ПҶпјӣиҜҒдёҚеҮәжқҘВ¬ПҶ\neg \varphiВ¬ПҶпјҢйӮЈе°ұдёҖе®ҡиғҪиҜҒеҮәжқҘПҶ\varphiПҶ

Lemma 9

Assume that ОҰ\PhiОҰ is consistent, negation complete, and contains witnesses. Then for all SSS-formulas ПҶ\varphiПҶ and ПҲ\psiПҲ:

(i)
ОҰвҠўПҶ\Phi \vdash \varphiОҰвҠўПҶ if and only if \Phi \not\vdash \neg \varphi.

(ii)
ОҰвҠў(ПҶвҲЁПҲ)\Phi \vdash (\varphi \vee \psi)ОҰвҠў(ПҶвҲЁПҲ) if and only if ОҰвҠўПҶ\Phi \vdash \varphiОҰвҠўПҶ or ОҰвҠўПҲ\Phi \vdash \psiОҰвҠўПҲ.

(iii)
ОҰвҠўвҲғxПҶ\Phi \vdash \exists x \varphiОҰвҠўвҲғxПҶ if and only if there is a term tвҲҲTSt \in T^StвҲҲTвҖӢSвҖӢвҖӢ such that ОҰвҠўПҶtx\Phi \vdash \varphi \frac{t}{x}ОҰвҠўПҶвҖӢxвҖӢвҖӢtвҖӢвҖӢ.
